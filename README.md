# APSO Ultimate - Aplikasi Surat Otomatis BP2JK Kalteng

## Deskripsi
Aplikasi berbasis Streamlit untuk membuat surat dinas otomatis sesuai standar TNDE.

## Fitur
- ✅ Standar TNDE (margin, font, spasi)
- ✅ Smart placeholder replacement
- ✅ Split-run handling untuk Word XML
- ✅ Preview dokumen
- ✅ Dynamic table injection
- ✅ Multi-signature support

## Instalasi
```bash
pip install -r requirements.txt
streamlit run app.py
